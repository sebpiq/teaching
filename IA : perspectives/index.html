Reconnaître un carré

Instructions pour
reconnaître un carré

  1. Compter le nombre de côtés → doit être égal à 4
  2. Mesurer la longueur de chaque côté → tous les côtés doivent être égaux
  3. Mesurer chaque angle → tous les angles doivent être de 90°
  4. Si toutes les conditions sont remplies → c'est un carré ✓
